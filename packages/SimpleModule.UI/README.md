# @simplemodule/ui

[![npm](https://img.shields.io/npm/v/@simplemodule/ui.svg)](https://www.npmjs.com/package/@simplemodule/ui)

Pre-built React UI component library for SimpleModule, built on Radix UI primitives with Tailwind CSS styling.

## Installation

```bash
npm install @simplemodule/ui
```

## Quick Start

```tsx
import { Button } from '@simplemodule/ui/components';
import { cn } from '@simplemodule/ui/lib/utils';

export function MyPage() {
  return <Button variant="default">Click me</Button>;
}
```

## Components

Accordion, Alert, AspectRatio, Avatar, Badge, Breadcrumb, Button, Calendar, Card, Chart, Checkbox, Collapsible, Command, Container, DataGrid, DataGridPage, DatePicker, Dialog, DropdownMenu, Field, Grid, HoverCard, Input, Label, PageHeader, PageShell, Popover, Progress, RadioGroup, Resizable, ScrollArea, Section, Select, Separator, Sheet, Sidebar, Skeleton, Slider, Spinner, Stack, Switch, Table, Tabs, Textarea, Toast, Toggle, ToggleGroup, Tooltip

## Key Features

- **48 components** built on Radix UI primitives
- **Tailwind CSS** styling with full theme customization
- **Accessible** -- ARIA-compliant via Radix UI
- **Tree-shakeable** -- import only what you use
