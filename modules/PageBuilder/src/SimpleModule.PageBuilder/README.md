# SimpleModule.PageBuilder

Visual page builder module for [SimpleModule](https://github.com/antosubash/SimpleModule) — a modular monolith framework for .NET.

## Features

- Visual page creation and management
- Page templates and tag support
- Publish/unpublish workflow
- 22 endpoints for full page lifecycle management
- Built-in page viewer interface

## Installation

```bash
sm install SimpleModule.PageBuilder
```

Or via .NET CLI:

```bash
dotnet add package SimpleModule.PageBuilder
```

## Usage

The module is auto-discovered by the SimpleModule framework. Navigate to the page builder section to create and manage pages.
