# SimpleModule.Marketplace

Module marketplace for [SimpleModule](https://github.com/antosubash/SimpleModule) — a modular monolith framework for .NET.

## Features

- Browse and discover available SimpleModule modules from NuGet
- Search, filter by category, and sort modules
- View module details, README, and version history
- One-click install commands (SM CLI and .NET CLI)
- Installed module detection
- Caching for fast browsing

## Installation

```bash
sm install SimpleModule.Marketplace
```

Or via .NET CLI:

```bash
dotnet add package SimpleModule.Marketplace
```

## Usage

The module is auto-discovered by the SimpleModule framework. Navigate to `/marketplace/browse` to explore available modules.
