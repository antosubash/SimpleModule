# Task: Bring source files under 300 lines

Goal: every `.cs`, `.ts`, `.tsx` file in the repo under 300 lines.

## Scope rules
- **Skip** auto-generated EF Core migrations (`**/Migrations/*.cs`, `**/Migrations/*.Designer.cs`) — regenerated by `dotnet ef`, hand-edits get erased.
- **Skip** CLI template files (`cli/SimpleModule.Cli/Templates/*.cs`) — mostly verbatim-string literals.
- **Skip** auto-generated `packages/SimpleModule.Client/src/routes.ts` — regenerated from endpoint constants.
- Verification: `dotnet build` (0W/0E), `npm run typecheck`, `npm run lint`.

## Refactor strategies
- **React pages** → extract sub-components into sibling `Pages/components/` folder; hooks into `.ts` files.
- **Services (C#)** → existing `partial class` pattern; split by responsibility (e.g. `Service.Export.cs`, `Service.Logging.cs`).
- **Non-partial C# classes** → add `partial` and split helpers/logging into sibling files.
- **UI library components** → split multi-purpose components into focused files (`chart.tsx` → `chart-tooltip.tsx` + `chart-legend.tsx` + `chart-context.ts`).

## Lessons captured
- Adding `using` directives is required when splitting partials that reference types the original file imported. The three initial service splits had missing usings (`Microsoft.EntityFrameworkCore`, `SimpleModule.Core`, `SimpleModule.Core.FeatureFlags`) — caught by `dotnet build` on second run.
- `packages/SimpleModule.Client/src/routes.ts` is auto-regenerated when the build runs and sometimes reformats line wraps. Revert it before committing to keep the diff clean.
- Enforcing under-300 going forward: consider adding a `wc -l` check to CI that blocks PRs introducing new files over 300 lines (exempting migrations, CLI templates, and auto-gen paths).

## CI check candidate
```bash
# fails if any tracked .cs/.ts/.tsx is > 300 lines
find . -type f \( -name "*.cs" -o -name "*.ts" -o -name "*.tsx" \) \
  ! -path "*/node_modules/*" ! -path "*/bin/*" ! -path "*/obj/*" \
  ! -path "*/dist/*" ! -path "*/wwwroot/*" ! -path "*/.git/*" \
  ! -path "*/Migrations/*" ! -path "*/cli/SimpleModule.Cli/Templates/*" \
  ! -path "*/packages/SimpleModule.Client/src/routes.ts" \
  -exec wc -l {} + | awk '$1 > 300 && $2 != "total" { count++; print }
     END { if (count > 0) exit 1 }'
```
